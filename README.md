# 📊 Hero MotoCorp Equity Research & Financial Modeling

## Overview

This project presents a comprehensive Equity Research analysis of **Hero MotoCorp Ltd.**, integrating financial statement modeling, forecasting, valuation, and investment analysis. The objective was to estimate the company's intrinsic value by building a structured financial model and evaluating its financial performance using industry-standard valuation methodologies.

The project also includes an interactive HTML dashboard for visualizing key financial metrics and insights.

---

## Project Objectives

* Analyze Hero MotoCorp's historical financial performance.
* Build an integrated 3-statement financial model.
* Forecast the company's financial statements (FY26E–FY30E).
* Estimate intrinsic value using Discounted Cash Flow (DCF) valuation.
* Compare valuation with listed peers using Relative Valuation.
* Develop an interactive dashboard to present financial insights.

---

## Financial Model Components

### Revenue Model

* Revenue segmented by:

  * Two-Wheeler Sales
  * Spare Parts
  * Other Operating Revenue
* Revenue forecasts based on historical growth trends and business assumptions.

### Three-Statement Financial Model

* Income Statement
* Balance Sheet
* Cash Flow Statement

All three statements are dynamically linked to ensure consistency across forecasts.

### Supporting Schedules

* Capital Expenditure Schedule
* Depreciation Schedule
* Working Capital Schedule
* Debt & Interest Schedule
* Other Supporting Assumptions

---

## Valuation Methodology

### Discounted Cash Flow (DCF)

The valuation model includes:

* Free Cash Flow to Firm (FCFF)
* Weighted Average Cost of Capital (WACC)
* Terminal Value Calculation
* Enterprise Value
* Equity Value
* Intrinsic Share Price Estimation

### Relative Valuation

Comparable company analysis using valuation multiples of peer companies including:

* Bajaj Auto
* TVS Motor Company
* Eicher Motors
